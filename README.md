# 🇮🇷 راهنمای فارسی Hermes Agent

ترجمه فارسی کامل داکیومنت رسمی [Hermes Agent](https://hermes-agent.nousresearch.com/docs/)

## درباره پروژه

این مخزن شامل ترجمه فارسی تمام مستندات Hermes Agent است. سایت GitHub Pages مشابه سایت اصلی با پشتیبانی کامل از راست به چپ (RTL) طراحی شده است.

## مشاهده سایت

🔗 **[مشاهده سایت](https://bookunt3d.github.io/hermes-agent-fa/)**

## ساختار پروژه

```
hermes-agent-fa/
├── _config.yml          # تنظیمات Jekyll
├── _layouts/            # قالب‌های صفحات
├── _includes/           # شامل‌ها (سایدبار و...)
├── assets/              # CSS و JS
├── docs/                # محتوای ترجمه‌شده
│   ├── getting-started/ # شروع سریع
│   ├── user-guide/      # راهنمای کاربر
│   ├── integrations/    # یکپارچه‌سازی‌ها
│   ├── guides/          # راهنماها
│   ├── developer-guide/ # راهنمای توسعه‌دهنده
│   └── reference/       # مرجع
└── scraped/             # محتوای اصلی (انگلیسی)
```

## مشارکت

اگر اشکالی در ترجمه پیدا کردید یا می‌خواهید کمک کنید، issue یا PR بزنید.
